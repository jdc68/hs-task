# Task (original)
Даны три целые числа a, b , c  больше 0. Есть два варианта   либо добавлять b к a (a+=b), либо добавлять a к b (b+=a).  В результате  получаются новые значение а  и b. Напишите программу на NODE JS, которая выведет YES или NO в зависимости от того можете ли получить  значение c,  добавляя  a к b  или b к a друг к другу.
